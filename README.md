 Student Management System
1:Overview
This is a Student Management System developed as a Programming Fundamentals (PF) semester project.
The project allows users to manage student records such as adding, viewing, updating, and deleting student information.
It is designed to demonstrate basic programming concepts like loops, arrays, functions, and file handling (if used).

2: Technologies Used
Programming Language:C++
IDE: Dev C++ 

3:Features
Add new student records
Display all students
Search student by ID / name
Update student information
Delete student record
Simple and user-friendly console interface
